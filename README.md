# 1C-A-system-for-automating-calculations-in-the-student-cafeteria

Конфигурация выполнена в 1С:Предприятие 8.3

Описание работы:

Есть список студентов, всем присваивается статус в зависимости от которого начисляется субсидия.
Студенты могут использовать субсидированные деньги только с 12 до 13 часов, в остальное время обед оплачивается за свой счет.
Если субсидия заканчивается, то тоже обед оплачивается за свой счет.

Всего 3 системы: 

Бухгалтерия - для начисления субсидированных денег студентам.

Касса - для проведения оплаты.

Администрация - там присутствует список студентов, все статусы и количество начисяемых денег. Так же там происходит присвоение статусов студентам
